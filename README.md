# FastAPI基礎セットアップ課題

API開発における重要な概念（バリデーション、エラーハンドリング、高度なレスポンス制御）を学習するためのバックエンドAPIプログラムです。<br>
現在はSwagger UIを通じてデータのやり取りが正しく行われることを確認する段階です。

「お店の自動受付システム」のような役割を持ち、ユーザーからのリクエストが正しいルールに従っているかをチェックし、適切な応答を返す仕組みをシミュレートしています。

---

## 実施内容

スクールのカリキュラムに沿って、以下を実装しました：

1. **基本エンドポイント**  
   GETリクエストによる疎通確認 (Hello World)

2. **クエリパラメータ**  
   パス/クエリパラメータによる動的データ処理

3. **データ検証 (Pydantic)**  
   リクエストボディのバリデーションと POST 実装

4. **エラーハンドリング**  
   使用クラス: `HTTPException`  
   → カスタムエラー（400エラー）を返す

5. **詳細なバリデーション**  
   使用クラス: `Query`  
   → 文字数制限（3〜50）や数値制限（1以上）

6. **レスポンスの多様化**  
   ステータスコード変更、HTML・テキスト応答の切り替え

7. **カスタムヘッダー**  
   使用クラス: `Response`  
   → 追加ヘッダーを付与可能

---

## 使用技術

- Python 3.x
- FastAPI
- Uvicorn (ASGI サーバー)
- Pydantic (データバリデーション)

## セットアップと起動方法

このプロジェクトをローカル環境で実行する手順です。

### 1. プロジェクトディレクトリへの移動

ターミナルまたはコマンドプロンプトを開き、プロジェクトのルートディレクトリ（このREADMEファイルがある場所）に移動します。

### 2. 仮想環境の作成

```bash
# Windows / Mac / Linux 共通
python -m venv venv

# ※上記が動かない（Mac/Linux等）場合はこちら
# python3 -m venv venv
```

### 3. 仮想環境の有効化

お使いのOSに応じて以下のいずれかを実行してください。

**Windows (PowerShell)**

```powershell
.\venv\Scripts\Activate.ps1
```

**Windows (コマンドプロンプト)**

```cmd
venv\Scripts\activate
```

**Mac / Linux**

```bash
source venv/bin/activate
```

### 4. 必要パッケージのインストール

```bash
pip install -r requirements.txt
```

### 5. 起動

```bash
uvicorn main:app --reload
```

### 6. 動作確認

ブラウザで以下にアクセスしてください：

👉 [http://127.0.0.1:8000/docs](http://127.0.0.1:8000/docs)

Swagger UI にて各エンドポイントの動作を確認できます。
